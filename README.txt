        「ピコピコ化計画」 by 片山博文MZ

    ◆◇◆◆◇◆◆◇◆◆◇◆◆◇◆◆◇◆◆◇◆◆◇

【概要】

    「ピコピコ化計画」は、パソコンの操作などに音声を加えます。

    例えば、マウスでどこかをクリックしたら音が鳴るとか、
    キーボードで何か打つと、音が鳴るといった効果を実現します。

【使い方】

    実行ファイル「PicoPico.exe」をダブルクリックすると、
    「ピコピコ化計画」のメイン ウィンドウが表示されます。

    音声ファイル（WAVファイル）を選び、「更新」ボタンを押すと、
    特定の操作に対して音声効果が表れます。

    PicoPico.exeと同じフォルダに拡張子が.wavの音声ファイルを
    置いてアプリを再起動すれば、追加の音声が選べるようになります。

    タスクバーに常駐できます。自動起動には対応していません。

    「終了」ボタンを押すとアプリが終了します。

【対応環境】

    - Windows XP 以降。

    「ピコピコ化計画」はフリーソフトです。無料で使用できます。
